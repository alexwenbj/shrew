About Code Shrew

Code Shrew is a platform for learning and practicing basic concepts of programming. In its heart is a programming environment where you can play with code by creating drawings and animations using a simple Python-like syntax. It is accompanied by an interactive course presenting a variety of fundamental programming concepts using the environment.

The tool and the site were built by Ludwik Trammer and all the lessons were created by Jamie Nunez.

Technology

We proudly use the following technologies (and many others):

  • Django (Python 3) - the server-side framework,
  • PostgreSQL - the database,
  • Bulma - the CSS framework,
  • Skulpt - the in-browser Python interpreter,
  • svg.js - the graphics library,
  • Font Awesome - the icons (both on the site and in the icons module),
  • Code Mirror - the code editor widget,
  • Cloud Flare - the CDN and caching provider,
