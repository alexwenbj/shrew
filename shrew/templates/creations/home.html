Welcome to Code Shrew!

Code Shrew is a platform for learning and practicing basic concepts of programming. In its heart is a programming environment where you can play with code by creating drawings and animations using a simple Python-like syntax. It will be accompanied by an interactive course presenting a variety of fundamental programming concepts using the environment.

The site will soon have much more features (including saving and sharing created drawings and animations). For now, we would like you to take a look at the interactive editor and a preliminary version of the lessons.

Mostly we want you to have fun. Please, don't leave the site before creating at least couple of animations! You may want to read through the documentation first, to see what's possible. You might also get inspired by some of the samples that we included below.

We are very interested in your feedback! Please, share with us any suggestions or ideas about Code Shrew that you might have. They might be about the interactive editor (i.e., the tool for drawing / animating with code), the lessons, or the general concept of the site. Bug reports would also be very appreciated. The thing that we would especially like to see is your creations - source code of drawings and animations that you've made with the site. Please share them with us!

But first and foremost let us know: was playing with the site any fun? :)
